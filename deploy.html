System Requirements

What You Need to Deploy

Required

  • Operating Systems
    Windows 10/11/Server 2019+, macOS 12+, Ubuntu 20.04+, RHEL 8+
  • Network Access
    Outbound HTTPS (443) to VIGILAX cloud intelligence endpoints
  • Admin Privileges
    Local admin or domain admin required for agent deployment
  • Memory
    Minimum 4GB RAM per monitored endpoint (8GB recommended)
  • Disk Space
    2GB for agent installation + log storage per endpoint

Optional / Enhanced

  • SIEM Integration
    Splunk, Microsoft Sentinel, IBM QRadar — optional but recommended for Archon+
  • Active Directory / Azure AD
    Enables zero-trust identity features on Warden+
  • Cloud Platform API Access
    AWS, Azure, GCP — enables cloud workload protection
  • MDM / EDM Solution
    Intune, Jamf — streamlines mobile and managed device enrollment
